# Skill: Fixture Project Factory

Crée un dépôt Git minimal, rapide et prédictible.

## Fixture de référence

Projet Python avec :

- `src/calculator.py` contenant une fonction volontairement incomplète ;
- `tests/test_calculator.py` ;
- `README.md` ;
- commit initial propre ;
- tâche : « implémenter `multiply(a, b)` et faire passer les tests ».

## Sorties

Chemin du dépôt, commit initial, branche, hash attendu du contenu, commande de test et patch attendu.

## Contraintes

Aucun accès réseau, aucune dépendance externe, tests en moins de cinq secondes, noms uniques par scénario.
