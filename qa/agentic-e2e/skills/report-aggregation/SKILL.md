# Skill: Report Aggregation

Fusionne les résultats, audits et findings sans perdre la chronologie.

## Calculs

- taux de réussite obligatoire/étendu ;
- première étape bloquante ;
- durée par étape ;
- nombre de défauts par gravité ;
- scénarios affectés ;
- flakes ;
- invariants de sécurité réussis/échoués.

Génère `findings.json`, `campaign.json` et `final-report.md` à partir des modèles.
