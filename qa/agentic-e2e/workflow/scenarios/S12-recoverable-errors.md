# S12 — Erreurs récupérables

Injecter séparément : executable de fixture absent, artefact agent invalide, code retour non nul, absence de modification entraînant `adapter_blocked`, chargement projet en erreur, export impossible et copie d'état tronquée.

Pour chaque cas, vérifier le message ordonné : cause, effet, préservation, preuve, récupération. Le TUI doit rester utilisable lorsque l'erreur est récupérable et les politiques produit ne doivent jamais être modifiées pour provoquer ou contourner l'erreur.
