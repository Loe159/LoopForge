# S10 — Multi-projet et même basename

1. Créer deux dépôts dans des parents différents avec le même nom de dossier.
2. Initialiser les deux.
3. Créer un run dans chaque projet.
4. Naviguer via Home/Ctrl+P.
5. Vérifier project ids, run roots, current run et preuves séparés.
6. Déplacer un projet puis l'ouvrir ; vérifier qu'aucune fusion silencieuse ne se produit.
